## Description
This project was build with Angular and TypeScript. the purpose of this application is to retrieve character data on all 
Harry Potter characters using the public REST API from https://hp-api.onrender.com/

## Features
- displays all characters from Harry Potter 
- Allows filtering characters by house
- Opening character cards to view their details

## Running locally
To start a local development server, run:
ng serve
Once the server is running, open your browser and navigate to `http://localhost:4200/`
